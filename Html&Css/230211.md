## Padding

### inline-block

패딩은 콘텐츠와 요소 테두리 사이의 거리를 정의.

인라인의 패딩은 다른 요소를 위로 이동시키지는 않는다.

인라인요소를 블럭으로 쓰려면 display: inline-block (인라인과 블럭 결합.

## 미디어 쿼리

### BreakPoint

스마트폰 : 480 픽셀

태블릿 : 768 픽셀

노트북 : 1,024 픽셀

데스크탑 컴퓨터 : 1,200 픽셀

1,200 픽셀을 넘어가면 모두 TV 화면에 맞춘다고 생각하면 된다.
